# 3D 打地鼠

一款基于 Three.js 的 3D 打地鼠游戏，浏览器直接运行。

## 访问地址

直接在浏览器打开 `index.html` 即可游玩。

在线版本：https://yeluo45.github.io/whack-a-mole-3d/

## 功能特性

- **真实 3D 场景**：草地、阴影、圆环地洞
- **可爱地鼠模型**：程序化生成，带眼睛、耳朵、鼻子、腮红
- **弹性动画**：地鼠冒出和缩回使用弹性缓动
- **击中特效**：圆形扩散击中效果
- **准星系统**：自定义准星，跟随鼠标
- **分数计时**：30秒倒计时，实时显示得分
- **开始/结束界面**：游戏状态管理

## 操作方式

- 移动鼠标瞄准地鼠
- 点击左键敲打地鼠
- 每个地鼠 +10 分

## 技术栈

- Three.js 0.160.0 (CDN)
- 原生 ES Module
- 单 HTML 文件，无构建依赖

## 部署

已配置 GitHub Pages，push 到仓库后自动部署到 https://yeluo45.github.io/whack-a-mole-3d/
